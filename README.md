# Описание
Данный репозиторий посвящен работе по прогнозу центров воздушного загрязнения (PM) в Европе методами машинного обучения. В нем содержатся датасеты и скрипты, реализующую работу метода.

# Запуск
Для запуска процесса используйте команду
```
python main.py *путь к файлу с датасетом* *путь к файлу с координатами*
```
